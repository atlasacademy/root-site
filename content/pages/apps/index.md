### [Atlas Academy API](https://api.atlasacademy.io)  
  
API Endpoints for serving FGO related data. Data is maintained and automatically updated within an hour of a new version of the game. See documentation for details.  
  
### [Atlas Academy DB](/atlas-academy-db/)  
  
Tool to navigate current game data for JP and NA FGO.

Index of drop locations and rates.
