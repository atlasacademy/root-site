<strong>TL;DR:</strong> According to the Domus Aurea drop data provided for Quetzmas, the increased gold currency efficiency from the Muscle Paradise node saves some AP over the Front Square node, resulting in a few extra boxes (minus the QP for reasons explained below) in exchange for a variable amount of QP.

<strong>SUPER TL;DR: DOING MUSCLE PARADISE INSTEAD OF FRONT SQUARE FOR GOLD CURRENCY IS LITERALLY EXTRA LOTTO BOXES (MINUS THE QP)</strong>

<em><strong>Important Considerations:</strong></em>

This method of squeezing out extra lotto boxes should NOT be used if the following scenarios apply to you:

1. You want hearts/fluids and don't need soaps.
2. You've already finished gold shop (duh).

<em><strong>Sample Scenario w/ Math:</strong></em>

<strong>Assumptions Made:</strong>

-Gold currency bonus is +1 (friend support CE).  
-Lotto currency bonus is +6 (friend support MLB CE included).  
-Total gold shop currency needed is 2200 (everything except for lotto CE and pieces/monuments).

<strong>Muscle Paradise Calculations:</strong>

The gold currency rate is 62.87 + (20.96 * X) drops/run, and each run requires 5 belts (2.5M QP).

Drops/run: 62.87 + 20.96 = 83.83  
Number of runs: 2200 / 83.83 = 26.24 (round up to 27)  
Number of belts needed: 27 * 5 = 135  
QP cost conversion of belts: 135 * 0.5 (500k QP/belt) = <strong>67.5M QP</strong>

Important note: this means that one would at least need to have 34 boxes first in order to obtain the belts necessary

<strong>Front Square Calculations:</strong>

The gold currency rate is 31.68 + (13.84 * X) drops/run.                                           The lotto currency rate is 34.73 + (13.16 * X) drops/run (here, X is assumed as +4; 4 lotto CEs are used with a friend gold CE).

Gold Drops/run: 31.68 + 13.84 = 45.52                                                                 Number of runs: 2200 / 45.52 = <strong>48.33 (round up to 49)</strong>

Lotto Drops/run: 34.73 + (13.16 * 4) = 83.37                                                       Number of drops: 83.37 * 49 = 4281                                                                       Number of boxes: 4281 / 600 = ~7                                                                                   QP gained: 7 * 6.48 = <strong>46.23M QP</strong>  
  
<strong>Final Calculations:</strong>

The lotto currency rate for lotto node is 44.51 + (16.56 * X) drops/run.

Drops/run: 44.51 + 16.56 * 6 = 143.87  
Number of drops: 143.87 * 49 = 7049.63  
Number of boxes: 7049.63 / 600 = 11.74 (rounded down to 11)  
QP per box: 11 * 6.48 = <strong>71.28M QP</strong>

<strong>Thus, you've spent</strong> <u><strong>42.4</strong></u><u>5</u><strong>M QP</strong> (67.5 +46.23 - 71.28) worth of belts on Muscle Paradise to save <strong>49</strong> runs of Front Square. This means you've gained extra 4 <strong>boxes worth of mats,</strong> MP<strong>,</strong> exp<strong>, etc</strong> (minus the QP)<strong>.</strong>
